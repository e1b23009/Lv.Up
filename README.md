# BLOCK ADVENTURE
コントローラーを用いて、制限時間内にゴールをめざす横スクロールアクションゲーム。
# 準備

### UnityHubのダウンロード
1. https://unity.com/ja/download からunityhubをインストールする。
2. UnityHubのInstallsタブのInstallEditerから2022.3.40f1を選択する。

### Unityプロジェクトのダウンロード
1. githubにてunityプロジェクトをダウンロードする。
2. UnityHubを開きエディタバージョン6000.2.9f1で当プロジェクトを開く。
3. お使いのWindowsOSに合わせて64bitか32bitでビルドを行う。

### BLOCK ADVENTUREのダウンロード
[BLOCK ADVENTUREのリポジトリ]
(https://github.com/e1b23009/Lv.Up )にある`Code`から`Download ZIP`を選択し、ダウンロード後解凍してください。解凍する場所はUnityHubです。

### アセットのダウンロード
1.	LV.UPの中にある「UniSideGame_Assets(LV.UP)」をダウンロード後解凍してください。
2.解答場所は先ほどダウンロードしたフォルダの中の「Assets」の中に解凍してください。

### コントローラーの接続
1. PCの設定内、デバイスからBluetooth接続を開く
2. JoyConのシンクロボタン(側面の押しにくいボタン)を長押し
3. PCのデバイス一覧からJoyConの左右を探し接続する


### 起動

1. UnityHubを起動し、「Add」をクリックすると、エディタが表示されるので、「ramendoukoukai_kitayamasai」を選択してください。
2.起動したら、上部にある「▶」再生ボタンをクリックするとゲームが起動します。

## Features
 
joyConのボタンを押す・振るといった基本操作で遊ぶ、1対1で戦う対戦ゲームです。

### Title画面
マウスカーソルかAボタンでSTARTボタンを押すことでゲーム画面に遷移する。

### game画面
※コントローラーからしか動かせない
1．移動
   左STICKを左右に傾けると移動
2. ジャンプ  
   Aボタンを押すとジャンプ、ダッシュ中はジャンプ力上昇
3. 射撃
   Bボタンを押すと射撃
4. ダッシュ
   RTを押している間ダッシュ
5. ブロック破壊
   破壊可能ブロックに接触中にLTを押すとブロックを破壊

### Result画面
タイトルに戻るボタンを押すとタイトル画面に、リスタートボタンを押すとゲーム画面に遷移する。
